# prm393-dart-tutorial

Repository dùng để thực hành và ghi chú quá trình học Dart trước khi chuyển sang Flutter.  
Nguồn học chính: tài liệu và tutorial từ trang chính thức của Dart.

## 📚 Mục tiêu

- Làm quen với cú pháp Dart
- Hiểu tư duy lập trình với Dart
- Nắm vững Dart core trước khi học Flutter
- Thực hành CLI app với Dart
- Làm quen với project structure và package management

## 🚀 Nguồn học

- Dart Official Learn: https://dart.dev/learn
- Dart Documentation: https://dart.dev/guides
- Dart Packages: https://pub.dev

# 🛠️ Environment Setup

## Cài Dart SDK

Download tại:

https://dart.dev/get-dart

Kiểm tra phiên bản:

```bash
dart --version
```

---

# ▶️ Run Dart Program

Chạy file Dart cơ bản:

```bash
dart run filename.dart
```

Ví dụ:

```bash
dart run hello.dart
```

---

# ▶️ Run CLI Project

Di chuyển vào thư mục `cli`:

```bash
cd cli
```

Run project:

```bash
dart run
```

---

# 🧪 Run Test

```bash
dart test
```
